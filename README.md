# GA4 Traffic Campaign Testing Checklist

A public checklist for teams validating analytics and controlled traffic workflows with [Traffic Creator](https://traffic-creator.com/).

## Checks

1. Confirm GA4 realtime receives expected sessions.
2. Validate country and device mix against campaign settings.
3. Check source/referrer attribution before larger marketing pushes.
4. Review landing-page engagement and funnel events.
5. Compare campaign pacing with expected daily volume.
